# Induction machine calculator
This Python project was developed in a group as part of practical work at my engineering school. It enables the simplified calculation of certain electrical parameters of an induction machine based on physical parameters.
It is provided as-is, open-source (GNU GPL v3 license), and will receive no further support, even if the results are incorrect (which they are).
No documentation is provided.

# Calculateur de machine asynchrone
Ce projet Python a été réalisé en groupe dans le cadre de travaux pratiques dans mon école d'ingénieur. Il permet le calcul simplifié de certains paramètres électriques d'une machine asynchrone à partir de paramètres physiques.
Il est fourni en l'état, en open-source (licence GNU GPL v3), et ne bénéficiera d'aucun support à l'avenir, même si les résultats sont incorrects (ce qui est le cas).
Aucune documentation n'est fournie.
